# Tabicat – Software Engineering Projekt

Dieses Repository enthält die Entwicklung unserer Website im Rahmen des Software-Engineering-Projekts.  
Der aktuelle Stand im **main-Branch** entspricht dem fertiggestellten Prototypen.

---

## Projektstruktur & Abgaben

Im Repository befinden sich folgende projektrelevante Abgaben:

### 1. Fertiggestellter Prototyp
Der **main-Branch** enthält den finalen Prototypen der Website.

### 2. Anforderungs- und Machbarkeitsanalyse
Zu finden im Ordner:

/docs

Enthalten sind:
- Anforderungsanalyse  
- Machbarkeitsanalyse  

### 3. Projektplan
Der Projektplan ist als **Roadmap** unter folgendem Bereich im Repository hinterlegt:

Projects → Roadmap

---

## Deployment

### Hosting
Die Website wird über einen kostenlosen Webserver bei **InfinityFree** gehostet.

### Deployment-Prozess
- Deployment erfolgt über einen FTP-Server.
- Die Dateien werden mithilfe von FTP auf den Webserver hochgeladen.
- Der aktuell als live-version veröffentlichte Stand entspricht dem finalen Prototyp aus dem main-Branch.

### Live-Version
Die aktuell deployte Version ist erreichbar unter:

https://tabicat.co.uk
