# User Guide
## How to add trips
1. Clone this repository ```git clone "url"```
2. Look in the archetypes folder and choose the trip you want to add
3. Copy the right trip type md into the corresponding content folder
4. If needed delete old trips (especially voyagers)
5. Fill out the markdown
6. Add pictures if needed in the assets/images folder
7. Commit the changes ```git commit -m "first commit"```
8. Push the changes ```git push```

Your changes are automatically deployed to the website


## How to delete trips
Delete the corresponding markdown file in the content folder


## General structure
- archetypes: template markdown
- assets: images, styling files
- content: all trip info in markdown format
- docs: documentation about this project
- layouts: all html files, which do the main layout for the page
- static: the logo and the php files
- tests: the js tests
